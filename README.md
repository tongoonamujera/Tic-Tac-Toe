> In this project, we were requested  build a Tic Tac Toe game. The main goal is to put into practice the main concepts of Object Oriented Programming. Particularly, classes and access to their attributes. ## Getting Started

Clone

```git
git clone https://github.com/tongoonamujera/Tic-Tac-Toe.git
```

To get a local copy up and running, follow these simple example steps.


1. Open a terminal window and enter `git clone https://github.com/tongoonamujera/Tic-Tac-Toe.git`
2. Make your new cloned directory your active directory
3. Currently, you are in the "main branch."
4. Change your current branch to Project-setup branch `git checkout Project-setup`
